Requirements
=
- nvm

Development
===========

- ```nvm use```
- check if node version is 6.9.4
- if not ```nvm install 6.9.4```
- ```npm install && npm start```

Go to ```http://localhost:4000/#/```

Build Web App
=============
```npm run build```

To test build in browser, from project root directory, run:

```node server.js```

Go to ```http://localhost:3000```

Build Desktop App
=================
```npm run dist```
